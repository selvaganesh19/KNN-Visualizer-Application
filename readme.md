Welcome to **KNN-Visualizer-Application**! This project provides an intuitive interface for exploring and visualizing the K-Nearest Neighbors (KNN) classification algorithm. ## ğŸš€ Introduction

K-Nearest Neighbors (KNN) is a simple, yet powerful machine learning algorithm used for classification tasks. The **KNN-Visualizer-Application** leverages Python, scikit-learn, Matplotlib, and Gradio to let users:

- Interactively adjust KNN parameters
- Visualize decision boundaries
- Analyze model accuracy
- Work with popular datasets like Iris

---

## âœ¨ Features

- **Interactive UI**: Powered by Gradio for easy parameter tuning and visualization.
- **Dataset Selection**: Built-in support for classic datasets (e.g., Iris).
- **Parameter Control**: Change the number of neighbors, test/train split, and more.
- **Visualization**: View decision boundaries and classifier outputs.
- **Accuracy Metrics**: Instantly see model performance.
